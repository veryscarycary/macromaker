## Tech Debt by Phase

### Phase 2 — Infrastructure Upgrade
1. **NATV-05 doc drift** — `newArchEnabled=false` in `android/gradle.properties` and `fabric_enabled: false` in `ios/Podfile`, but REQUIREMENTS.md description reads "New Architecture enabled (`newArchEnabled=true`)". Intentional: Phase 3 squash commit e7530d3 reverted New Architecture due to react-native-screens 4.x bridge crash — decision documented in STATE.md. REQUIREMENTS.md should be updated to reflect "bridge-compat mode, new architecture deferred to v2."
2. **JSDP-04 latent risk** — `babel.config.js` has no plugins array; `react-native-worklets/plugin` not present. Reanimated 4.2.2 installed but worklet transforms not wired. No Reanimated-using source code found — latent risk. When Reanimated animations are added, Babel plugin must be restored.

### Phase 1 — Dependency Audit
3. **d3-shape undeclared** — `MacroGraph.tsx` imports `d3-shape` which is a transitive dep of `d3@5.15.1`, not declared in `package.json`. Add `"d3-shape": "^2.1.0"` (matching the transitive version) to lock it explicitly.

### Phase 3 — RN Upgrade
4. **cardStyle deprecated** — `navigation/index.tsx:61` uses `cardStyle` on `MenuModal` screen options, deprecated in React Navigation v7. Functional but should migrate to `contentStyle`.

### Phase 4 — Onboarding UX Polish
5. **ONBR-04 age unit** — Age input field has no unit label. Requirement wording ("lbs", "ft", "in") is ambiguous — age may be out of scope. Human verification passed. Low priority.
